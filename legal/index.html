    Data usage

    Hextech Friends only transfers your summoner name, summoner Id, profile icon and level to the relay server. The data will only be stored on the relay for the time your lobby is open. After this the data is immediately deleted. Additionally the data is not stored in any database, just the application memory, which will be cleared if the application restarts, even if you have a lobby open at that time.
